# Tech Fox Blog 
##### blog website built by laravel framework .

## Key Features

### Admin: 
  * Add /delete /update Section 
  * Add /delete /update user 
  * Add /delete u/pdate posts 
  * Change users rule /user / editor 
   --------------------------------------------------------
### Editor:
  * Add posts
  * Update posts
  * Delete posts

    ------------------------------------------------------
### Registered user:
 * Add comment 
 * View comments 

   ----------------------------------------------------------
### Normal user :
  * View posts only
